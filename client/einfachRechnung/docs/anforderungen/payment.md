### Register Payment

- User öffnet Rechnung
- User wählt "Zahlung erfassen"
- User erfasst Zahlungsbetrag
- Zahlungsdatum wird gesetzt
- Zahlung speichern
- Offener Betrag neu berechnen
- Falls vollständig bezahlt:
    - Rechnung erhält Status "paid"
- Falls teilweise bezahlt:
    - Rechnung erhält Status "partial"
