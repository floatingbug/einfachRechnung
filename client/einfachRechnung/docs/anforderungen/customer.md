### Create Customer

- User startet "Neuer Kunde"
- User erfasst Kundendaten
- Kundendaten validieren
- Prüfen ob Kunde bereits existiert
- Kunde speichern
- Kunden-ID erzeugen
- Kunde erscheint in Kundenliste


### Search Customer

- User öffnet Kundenliste
- User sucht Kunde über Name oder Email
- Passende Kunden werden gefiltert
- User wählt Kunde aus
- Kundendetails anzeigen


### Edit Customer

- User öffnet Kundendetails
- User wählt "Bearbeiten"
- User ändert Kundendaten
- Kundendaten validieren
- Änderungen speichern
- Kundendaten aktualisieren
